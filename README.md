## How to execute

Clone the project

```bash
  git clone git@github.com:enicio/hapu-landpage.git
```
Change to root directory

```bash
  cd hapu-landpage
```
install the dependencies

```bash
  yarn install
```

Start

```bash
  yarn start
```

```bash
  In your browser http://localhost:3000/
```
